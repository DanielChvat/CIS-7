# CIS-7

## Course Description

This course is an introduction to the discrete structures used in Computer Science with an emphasis on their applications. 

- **Functions**
- **Relations**
- **Sets**
- **Basic Logic**
- **Proof Techniques**
- **Basics of Counting**
- **Graphs**
- **Trees**
- **Discrete Probability**
