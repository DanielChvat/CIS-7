### Encryption

Let the plaintext character at a given index be denoted by ${P_i}$, the corresponding key character be denoted by ${K_i}$ and the encrypted character be denoted by ${E_i}$. ${E_i} is found using the following formula:

${E_i} = {(P_i + K_i) \mod 26}$
